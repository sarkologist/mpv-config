# Agent Policy

- Prefer right-hand home-row seek bindings for navigation: `j k l ;` for base jumps and `J K L :` for amplified jumps.
- Keep held-key repeat for home-row seeking in the ZSA Voyager mpv smart layer; plain `input.conf` text-key repeat only works when the keyboard sends normal repeated key events.
- Keep arrow and wheel seek bindings as fallback, not primary navigation.
- If remapping seek keys in the future, preserve this shape: small back/forward on the center keys and larger jumps on the outer keys.
